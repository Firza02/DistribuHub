# DistribuHub

Web-based distributor management system built using Laravel and MySQL.

## Features

- Dashboard Reporting
- Product Management
- Customer Management
- Invoice Management
- Transaction Recording
- Inventory Monitoring

## Tech Stack

- Laravel 12
- PHP
- MySQL
- Bootstrap
- JavaScript

## Installation

Clone repository:

git clone https://github.com/Firza02/DistribuHub.git

Install dependencies:

composer install

Setup environment:

cp .env.example .env

Generate key:

php artisan key:generate

Run migration:

php artisan migrate

Start server:

php artisan serve
